CoahuilOS

¡Bienvenido a CoahuilOS, el sistema Linux para el estado de Coahuila de Zaragoza!

¿Qué es CoahuilOS?

CoahuilOS es una distribución de Linux diseñada específicamente para ser un todo en uno las para los usuarios en el estado de Coahuila desde equipos educativos hasta computadoras de escritorio. Basada en Debian, CoahuilOS ofrece una experiencia de usuario estable, segura y fácil de usar incluso si nunca tocastes un sistema Linux o Windows anteriormente.

Características principales

  • Interfaz amigable y personalizable
  • Amplia selección de software preinstalado
  • Soporte para hardware local
  • Actualizaciones regulares y soporte comunitario
  • Enfoque en la educación y el desarrollo local
  • Compatibilidad con flatpak y Debian/Ubuntu

Descarga e instalación

Puedes descargar CoahuilOS desde nuestro sitio web oficial o también desde el repositorio oficial de Github. Ofrecemos imágenes ISO para diferentes arquitecturas, así como instrucciones detalladas para la instalación.
